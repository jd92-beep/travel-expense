📧 Email 自動匯入（免費 · 全自動）

推薦

Forward 確認 email 去你 Gmail → Google Apps Script（免費）用 Gemini 解析 → 自動入 Notion → app 顯示「⏳ 待確認」俾你一 tap 確認

🚀 點樣用？

  1. 喺 Gmail filter 設定：subject 含 #expense → label travel-expense
  2. Forward 任何 confirmation email 去你自己 Gmail，主旨加 #expense
  3. 5 分鐘內：Notion 會自動出現「⏳ 待確認」嘅 entry
  4. App 打開 History tab → 黃色卡片 → 按「✅ 確認」即可

⚙️ 一次性設定 (~3 min)：

  1. 點「📋 複製 Apps Script 代碼」
  2. 點「🔗 開 Apps Script 編輯器」→ "New Project"
  3. 貼入代碼，填入 Gemini key / Notion token / DB ID（app 同一套）
  4. 點 Run → 揀 setup → 授權 Gmail + 網絡 access
  5. Gmail 設 filter，完成！

✅ 之後每 5 分鐘自動 scan 一次，零人手操作。
