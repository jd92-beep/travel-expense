# Admin Console — 替換 Dashboard Tab

## 📋 概述
將現有 Dashboard tab 改造成 Admin Console，只限 Boss (`isBoss()`) 可見。數據來源用現有 `admin-kanban` Edge Function 嘅 `/api/snapshot` endpoint。

## ✅ Plan

### Phase 1: Tab 系統改造
**Files**: `app-compact/src/lib/tabs.ts`, `app-compact/src/App.tsx`, `app-compact/src/components/Shell.tsx`

1. `tabs.ts`: TabId 加 `'admin'`，`TAB_MANIFEST` 加 admin tab（label: 'Console'）
2. `App.tsx`: Admin tab 只喺 `isBoss(userEmail)` 時顯示
3. `Shell.tsx`: Admin tab icon 用 `Shield` 或 `LayoutDashboard`

### Phase 2: Edge Function API Client
**File**: `app-compact/src/lib/adminApi.ts` (新建)

```typescript
// Call admin-kanban Edge Function
const ADMIN_BASE = 'https://fbnnjoahvtdrnigevrtw.supabase.co/functions/v1/admin-kanban';

export async function fetchAdminSnapshot(range = '7d'): Promise<AdminSnapshot>
export async function deletePreview(userId: string): Promise<DeletePreview>
export async function deleteUser(userId: string, confirmPhrase: string): Promise<void>
export async function amendReceipt(receiptId: string, updates: Partial<AmendReceiptPayload>): Promise<void>
```

認證：用 Supabase session Bearer token。

### Phase 3: Admin Console Tab
**File**: `app-compact/src/tabs/Admin.tsx` (新建)

6 個 sections（cards）：

#### 3.1 用戶管理
- 用戶列表表格：email(masked)、joined、lastSeen、tripCount、receiptCount、health
- 每行可展開查看詳情
- 刪除用戶按鈕（需 confirm）

#### 3.2 旅程管理
- 旅程列表表格：name、owner、destination、dateRange、receiptCount、active/archived
- 點擊展開查看收據摘要

#### 3.3 收據管理
- 收據列表表格：store、amount、currency、date、status、notionSynced
- 按 trip 篩選
- Admin 修改按鈕（amend）

#### 3.4 Backend 健康監控
- Supabase 狀態 + RLS 狀態
- Notion 連接狀態（connectedUsers、syncedReceipts、failedJobs、pendingJobs）
- Broker health（5 個 provider 狀態）

#### 3.5 AI Provider 監控
- Provider 列表：name、status、model、lastTested、errors24h
- 用量統計（from usage_events）

#### 3.6 審計日誌
- Audit events 列表：action、target、timestamp
- 最近 20 筆

### Phase 4: Dashboard 保留功能遷移
將 Dashboard 嘅核心功能（trip switcher、budget card、today itinerary）移到其他 tabs 或 Settings：
- Trip switcher → Shell header（已有）
- Budget card → Stats tab
- Today itinerary → Timeline tab
- Recent expenses → History tab

### Phase 5: 樣式
沿用現有 Settings tab 嘅 card/accordion 風格，加 admin-specific 顏色（深色 header、紅色危險操作）。
