# Plan: Add Notion Connection to Admin Console

## 📋 問題
Admin Console 顯示 "Not Configured" 因為冇用戶連接過 Notion。需要喺 Console 加連接功能。

## ✅ Plan

### Phase 1: Edge Function — 加 connect-notion endpoint
**File**: `supabase/functions/admin-kanban/index.ts`

加新 endpoint `POST /api/connect-notion`:
- 參數: `{ userId, notionToken, notionDatabaseId }`
- 呼叫 Credential Broker `/integrations/notion/connect` 代用戶連接
- 儲存到 `integrations` table

### Phase 2: Admin API — 加 connectNotion function
**File**: `app-admin-kanban/src/lib/adminApi.ts`

加 `connectNotion(session, userId, token, databaseId)` function

### Phase 3: Admin UI — 加連接 Notion 按鈕
**File**: `app-admin-kanban/src/App.tsx`

喺 UserDetailsPanel 嘅 Connection Status section 加:
- "Connect Notion" 按鈕
- Modal: 輸入 Notion token + database ID
- 成功後更新 notionStatus 顯示

## 🔍 Verification
1. Build + deploy
2. Smoke test: 登入 → click user → 見到 Connect Notion 按鈕
3. 輸入 test credentials → 確認連接成功
