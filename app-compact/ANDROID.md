# Compact Android Shell

This Android shell is intentionally isolated from the live Compact web app.

## Safety Rules

- Work from `codex/android-compact-shell`, not `main`.
- Do not merge or push to `main` until the live web app has a safe deploy window.
- Android packaging must not change Compact routes, UI, Supabase, Notion, or broker logic.
- `android/app/src/main/assets/public` is generated by Capacitor and ignored by git.

## Commands

```bash
cd app-compact
npm run typecheck
npm run build:root
npm run android:sync
npm run android:debug
```

The debug APK is created at:

```text
app-compact/android/app/build/outputs/apk/debug/app-debug.apk
```

For release testing later:

```bash
cd app-compact
npm run android:bundle
```

The release AAB still needs proper signing before Play Store upload.

## Current Native Scope

- Bundled Capacitor Android shell.
- App id: `com.ftjdfr.travelexpensecompact`.
- Version: `0.8.0` / versionCode `800`.
- Permissions: internet, camera, image library access.
- Android backup disabled because the app contains travel expense data.
